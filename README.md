# 🐍 Snake Game – Modular Python Implementation

## 💡Overview

This project is an implementation of the classic Snake game developed as part of the course INF100 – Introduction to Programming at the University of Bergen. The game is built using the **uib_inf100_graphics** framework, an educational graphics library provided in the course environment. The objective is to control a snake that moves across the board, collects apples to grow, and avoids collisions with itself.


## 🧠 Technical focus

Although developed in an introductory programming course, the project emphasizes:
- Real-time game loop logic
- Event-driven keyboard input handling
- State management
- Collision detection logics
- Modular program structure across multiple files
- Basic automated test validation

The project focuses on writing structured and maintainable code rather than just achieving visual functionality.

## 🖼️ Framework

The project uses:
	•	Python 3
	•	uib_inf100_graphics

uib_inf100_graphics is a lightweight educational framework that provides:
	•	Window management
	•	Drawing primitives
	•	Keyboard event handling
	•	Timed update loops

It simplifies graphical programming while allowing focus on program structure and logic. The graphics was not the main focus.
