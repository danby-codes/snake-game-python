# 🐍 Snake – Real-Time Game Implementation in Python

⸻

## 💡 Overview

This project is an implementation of the classic Snake game developed as part of the course INF100 – Introduction to Programming at the University of Bergen.

The game is built using the uib_inf100_graphics framework, an educational graphics library provided in the course environment.

The objective is to control a snake that moves across the board, collects apples to grow, and avoids collisions with itself.

⸻

## 🧠 Technical Focus

Although developed in an introductory programming course, the project emphasizes:

	•	Real-time game loop logic
  
	•	Event-driven keyboard input handling
  
	•	State management
  
	•	Collision detection logic
  
	•	Modular program structure across multiple files
  
	•	Basic automated test validation


The project focuses on writing structured and maintainable code rather than just achieving visual functionality.

⸻

🖼️ Framework

The project uses:

	•	Python 3
  
	•	uib_inf100_graphics


uib_inf100_graphics is a lightweight educational framework that provides:

	•	Window management
  
	•	Drawing primitives
  
	•	Keyboard event handling
  
	•	Timed update loops
  

It simplifies graphical programming while allowing focus on program structure and logic.

The graphics layer was not the primary focus of the project.

📊 Assignment Grading

The project was manually graded according to the following rubric:

	•	20 points – Fully functional implementation without obvious errors
  
	•	10 points – Functional, but with bugs
  
	•	5 points – Good effort, but not fully functional
  
	•	0 points – Insufficient implementation
  

Additional points were available for extended functionality beyond the core requirements.


My final result: 20/20 points ✅
