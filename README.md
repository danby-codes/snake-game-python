Snake Game – Modular Python Implementation

Overview

This project is an implementation of the classic Snake game developed as part of the course INF100 – Introduction to Programming at the University of Bergen.

The game is built using the uib_inf100_graphics framework and focuses on structured program design, real-time logic, and modular code organization.

The objective of the game is to control a snake that moves around the board collecting apples while avoiding collisions with itself or the boundaries.
